## 语言/动作识别记号 (voiceProfile)

> 这一段是 Writer 写本角色任何对白 / 内心戏前**必须对照**的识别记号清单。每条要具体到**能在文本里直接命中**的程度——不要写"她说话很温柔"这种模糊描述（无法验证）；要写"句末偏好用'罢了'软化语气，且很少用感叹号"（可以扫词）。
>
> 主角必须填全 7 项；次要角色至少前 3 项（句长偏好 / 口头禅 / 称呼习惯）。空缺项用 `_未定_` 占位，不允许删字段——schema 完整性比内容多寡重要。

- **句长偏好**：（短促 / 中等 / 偏长，并写明原因——情绪 / 教养 / 身份。例：偏长且多分句，反映文人出身的迂回）
- **口头禅 / 语气助词**：（最多 3 个，要具体到字。例："嗯"、"罢了"、"是吗"。不要写"喜欢用语气词"这种笼统描述）
- **称呼习惯**：（自称 / 他称 / 对不同身份的人怎么切换。例：自称"我"，对长辈称"先生"，对同辈直呼本名不加敬称）
- **句式倾向**：（是否爱反问 / 命令式 / 省略主语 / 特定句首套话。例：常用"你应当 X"类祈使，少用感叹）
- **不会说的话**：（雷区台词；触线即出戏。例：绝不用市井语"哎哟""不就是"；绝不主动夸赞他人）
- **肢体识别动作**（次要角色可写 `_未定_`）：（一两个高频小动作。例：紧张时摩挲右手食指、不耐烦时短促清嗓）
- **沉默时的姿态**（主角必填；次要角色可写 `_未定_`）：（不说话时身体语言告诉读者什么。例：左手攥紧腰带 = 压抑怒气；目光斜下 = 自我否定）

**Writer 使用规则**：本章只要该角色开口或有内心戏，必须把这一段当作硬约束对照写——不是装饰参考。Auditor dim 1（OOC）会回查："这段对白有没有违反不会说的话清单 / 用了句长偏好相反的句子"。详见 [05-writer.md §13](phases/05-writer.md) 与 [09-auditor.md](phases/09-auditor.md)。
